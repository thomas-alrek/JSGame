(static) clamp(value, min, max) → {number}

Clamps a number into a range If the input value is bigger than max, the number is truncated to max. If the input value is smaller than min, the number is truncated to min. E.g Math.clamp(370, 0, 360) === 360;
Parameters:
Name Type Description
value number Number to clamp in range
min number Minimum range value
